<Info>
  **Prerequisite** You should have installed Node.js (version 18.10.0 or
  higher).
</Info>

Step 1. Fork the [FDAi repository](https://github.com/FDA-AI/FDAi) to your GitHub account.

Step 2. Clone the forked repository to your local machine.

Step 3. See the main README or the README's in any of the apps folders for instructions on how to run the app locally.
